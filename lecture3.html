What is Table tag ?

  • A table is used to arrange data into rows and columns.
  • Created using < table> tag.
  • < table> : defines the table.
  • < tr> : table row.
  • < td> : table data (cell).
  • < th> : table header (bold, centered by default).
  • border="1" : adds border (allowed since itâ€™s an HTML attribute, not CSS).
  • < caption> : tag is used to give a title/heading to a table.
  • < t foot> : Footer section (like totals or summary).
  1. colspan="n" merge n columns.
  2. rowspan="n" merge n rows.
