Concept of hyperlink

  • A Hyperlink is a text, image, or element that you can click to jump to another page, section, or resource.
  • It is created using the (anchor) tag with the href attribute.

What is HTML(hypertext markup language) ?

  1. HyperText → Text that has links (hyperlinks) to jump from one page to another.
  2. Markup Language → A way to structure and format content using tags

Understanding Tags

  • Tags are keywords inside angle brackets <>.
  • They tell the browser how to structure or display content.
  • Most tags come in pairs: opening and closing .
  • Some are self-closing (like < img>, < br>).
Some Most common tags are !
< h1> to < h6> Headings (big to small)
< p> Paragraph
< a> Hyperlink (with href)
< img> Image (self-closing)
< br> Line break (self-closing)

What is Heading tag ?

  • In HTML, heading tags are used to define titles or headings on a webpage.
  • They range from < h1> (largest) to < h6> (smallest).
  • Search engines (like Google) also read these to understand your content better (SEO).

  1. Main Title of Page

  2. Section Title

  3. Sub-section Title

  4. Smaller heading

  5. Even smaller
  6. Tiniest heading

What is Paragraph Tag ?

  1. < p> tag is used to write paragraphs of text in HTML.
  2. It automatically adds some space (margin) before and after the text.
  3. It is a block-level element (means it takes the full width).

What is Element in HTML ?

HTML Element: A complete structure made of a tag (opening + closing) and the content inside it.

What is Horizintal Line ?

  • Horizontal line = A line used to separate content in a webpage.
  • It is a self-closing tag (no ending tag).
    • Example

    This is first paragraph.

    This is second paragraph after a horizontal line.

What is Line Break

  • Line Break = moving to next line without horizontal line visible.
  • It is a self-closing tag (no closing needed).

Introduction to LIST !

A list in HTML is used to display items in an organized way (one after another).

  1. Ordered List (< ol>) – numbered list
    1. Apple
    2. Banana
    3. Mango
  2. Unordered List (< ul>) – bullet points
    • HTML
    • CSS
    • JavaScript

What is Anchor Tag ?

  • The anchor tag < a> is used to create hyperlinks in HTML.
  • It connects one page to another, or one section to another.
  • The href attribute defines the link’s destination.(hyperlink refrence)

What is Comment in HTML ?

  • Comments are notes in your code that the browser ignores.
  • Used to explain code or for reminders.

What is Attributes in HTML

  • Attributes give extra information about a tag.
  • They are always written in the opening tag.
  • Format: name="value"

What is Image tag in HTML?

  • The < img> tag is used to display images on a webpage.
  • It is a self-closing tag (no ending tag).
  • Needs attributes to work properly.
    • Important Attributes
      1. src → source (path or URL of the image)
      2. alt → alternative text (shown if image doesn’t load, also good for SEO)
      3. width / height → size of the image (in pixels)
