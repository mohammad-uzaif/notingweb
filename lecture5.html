What is an HTML Form ?

  • An HTML form is used to collect input/data from users.
  • Data can be anything → text, password, email, file, checkbox, etc.
  • Data from a form is usually sent to a server for processing (like login, signup, search).

Form tag :

  1. < form> is a container tag used to create an HTML form.
  2. It holds all the input elements (like text fields, radio buttons, checkboxes, dropdowns, etc.) where users can enter data.
  3. When submitted, it sends the data to a server or another page.

Important Attributes of < form>

  1. action → URL where form data will be sent.
  2. method → defines how data is sent:
    • GET → data visible in URL (for search, small data).
    • POST → data hidden in body (for login, signup, sensitive info).
  3. name → gives form a name (used in JavaScript).
  4. target → where to open result (_self, _blank).

Input tag :

  • < input> is an inline form element used inside form.
  • It is used to take user input (like text, number, password, email, file, etc.).
  • It is used to take user input (like text, number, password, email, file, etc.).

Important Attributes of < input>

  1. type → defines kind of input field.
    • text → single line text
    • password → hidden text (****)
    • email → email input
    • number → numeric input
    • radio → single-choice option
    • checkbox → multiple-choice option
    • file → upload files
    • submit → button to submit form
    • reset → button to reset form
  2. name → name of field (used when sending data).
  3. value → default value inside input.
  4. placeholder → hint text inside box.
  5. required → makes input mandatory.

What is label Tag in HTML?

  • < label> is used to describe a form input.
  • It improves accessibility → screen readers can read it.
  • Clicking on the label automatically focuses the related input.

Key Points

  • Associated with input using for attribute → matches the input’s id.
  • Makes forms user-friendly and accessible.
  • Can wrap input element directly instead of using for.

Name & Value Attribute in HTML Forms

  1. name Attribute
    • Purpose: Identify form data when sending to the server.
    • Each input that you want to send must have a name.
    • Server uses this name as a key to get the value.
  2. value Attribute
    • Purpose: Holds the default value of an input field or the value sent for buttons.
    • For text → default text in the box.
    • For submit button → text on the button.

What is Radio Button in HTML ?

  • Radio button is an input type used in forms.
  • Allows user to select only one option from a group.
  • Always used in groups with the same name attribute.

Key Points

  1. Only one radio button in a group can be selected at a time.
  2. value attribute defines what gets sent to the server when selected.
  3. checked attribute → makes a button selected by default.

For Attribute in HTML

  • The for attribute is used inside a < label> tag.
  • It links the label to a form input element (like text box, radio, checkbox).
  • Value of for must match the id of the input.

Why use it?

  1. Improves usability & accessibility.
  2. Clicking on the label text also selects or focuses the input.

Setting Radio Button Value

  • In radio buttons, name attribute groups options together.
  • value attribute tells which option was selected when the form is submitted.
  • Only one option from the same name group can be selected at a time.

Note :

  • name = groups radio buttons.
  • value = answer that is sent when that option is selected.

What is Checkbox?

  • Checkbox allows a user to select one or more options from a list.
  • Unlike radio buttons (only one choice), checkboxes can be multiple selected.

Key Points

  1. < input type="checkbox"> hota hai.
  2. name attribute → identifies group of checkboxes.
  3. value attribute → tells which option is selected when submitting form.
  4. Multiple checkboxes with the same name → multiple values submit ho sakte hain.

Placeholder

  1. placeholder is an attribute used inside form inputs.
  2. It shows a short hint or example text inside the input box before the user types anything.
  3. As soon as the user types, the placeholder disappears.

What is required Attribute?

  • required is a boolean attribute in HTML forms.
  • It makes an input field mandatory → user cannot submit the form without filling it.
  • If the field is empty, browser automatically shows a validation message.
