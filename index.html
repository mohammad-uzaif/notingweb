What is Internet ?

The Internet is a global network that connects millions of computers and devices to share information and communicate.

What is IP address ?

An IP address (Internet Protocol address) is a unique number assigned to each device on a network that identifies it and allows it to communicate with other devices over the Internet.

How to access a website ?

You can access a website by typing its URL (like www.google.com ) into a web browser, which then uses DNS to find the site’s IP address, sends a request to the web server, and displays the webpage on your screen.

Tap arrow to open ! HOW TO ACCESS A WEBSITE ?
  1. You type the website name
  2. Browser asks: “Where is this website?”
  3. DNS helps find the address
  4. Browser sends a request to the server
  5. Server replies back with the website
  6. Browser shows you the page

Who assign the IP address ?

Your ISP assigns your IP address, but the authority comes from IANA(Internet Assigned Numbers Authority)/ICANN(Internet Assigned Numbers Authority).

What is DNS?

DNS (Domain Name System) is like the phonebook of the Internet.

  • It converts domain names (like www.google.com) into IP addresses (like 142.250.190.78).
  • Without DNS, you’d have to remember numbers instead of names to visit websites
Tap arrow to open ! Consept of DNS ?
  1. You type a domain name in the browser (example: www.google.com).
  2. DNS Resolver (usually from your ISP) checks if it knows the IP.
  3. If not, it asks the Root DNS Server.
  4. The Root server points it to a TLD server (for .com, .org, .in, etc.).
  5. The TLD server points to the Authoritative DNS server for that domain.
  6. The Authoritative server replies with the IP address of the website.
  7. Now your browser can contact the server directly and load the site.

DNS = Translator between human-friendly names and machine-friendly IPs.

What is a Port Number?

A Port Number is like a door or channel on a computer that is used to send and receive specific types of data over the Internet.

  • An IP address identifies which device to talk to.
  • A Port number identifies which service or application on that device to talk to.
Tap arrow to open ! In Short

A Port Number is a unique number (0–65535) used along with an IP address to direct Internet traffic to the right application or service.

What is HTTP?

HTTP (HyperText Transfer Protocol)

  • It is the protocol (set of rules) used for transferring data (like web pages, images, videos) between a web browser and a web server.
  • Example: http://www.example.com
  • Data is sent in plain text (not secure).

What is HTTPS?

HTTPS (HyperText Transfer Protocol Secure)

  • It is the secure version of HTTP.
  • Uses SSL/TLS encryption so that data (like passwords, bank details) is safely transmitted.
  • Example: https://www.example.com
  • Protects against hackers and data theft.

IPv4 vs IPv6 ?

IPv4 (Internet Protocol version 4) :

  • The older version of IP.
  • Uses 32-bit address → about 4.3 billion unique addresses.
  • Written in dot-decimal format: (for Example: 192.168.0.1)
  • Problem: Because of the huge growth of the Internet, IPv4 addresses are running out.

IPv6 (Internet Protocol version 6)

  • The new version of IP.
  • Uses 128-bit address → provides a huge number of addresses (enough for every device in the world many times over).
  • Written in hexadecimal with colons:(Example: 2001:0db8:85a3:0000:0000:8a2e:0370:7334)
  • Has built-in security features and faster routing compared to IPv4.

What is a MAC Address?

  • MAC (Media Access Control) Address is a unique hardware ID given to a device’s network card (NIC) by the manufacturer.
  • It is used to identify devices within a local network (like WiFi, LAN).
  • Unlike IP, which can change, the MAC address is permanent (burned into the hardware).

What is LAN?

LAN (Local Area Network): A small network that connects devices within a limited area (like home, office).

Switch & Router :

Switch: Connects devices inside a LAN and forwards data to the correct device using MAC addresses.

Router: Connects your LAN to other networks (like the Internet) using IP addresses.

MERN STACK ?

MERN is a popular set of technologies used for Full-Stack Web Development.

M E R N stands for
M – MongoDB
  • A NoSQL database (stores data in JSON-like documents).
  • Example: Stores user info, posts, products.
E – Express.js
  • A backend framework for Node.js.
  • Helps create APIs and handle requests easily.
R – React.js
  • A frontend library for building interactive UIs.
  • Used to make dynamic websites and single-page apps.
N – Node.js
  • A runtime environment that lets you run JavaScript on the server side.
  • Executes backend code, connects to databases, etc.
